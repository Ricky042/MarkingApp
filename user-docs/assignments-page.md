# Assignments Page

Use the Assignments system to view, manage and mark assignments effectively.

---

## Assign Marks

1. Select an assignment from the **Assignments Tab**.  
2. Click **Mark Control Papers** to open the marking viewer.  
3. View and input marks into the rubric criterion and parallel marking system
4. Press confirm to save your marks

*MarkingApp automatically calculates averages and grade distributions.*

---

## Compare Grades Between Tutors

1. Select an **Assignment**.  
2. Once tutors have marked, it shows and compares marks between them to the standard.
3. **Colour Coding** displays the differentials for an easy comparison at a glace.
